# Einundzwanzig-Meetup-Website

Diese Webseite ist eine **Open-Source-Vorlage** für Bitcoin- und Einundzwanzig-Meetups.

Jedes Meetup ist herzlich eingeladen, dieses Repository zu **forken**, anzupassen und selbst zu hosten (z. B. über GitHub Pages). Verbesserungen, Design-Anpassungen und neue Features sind ausdrücklich erwünscht!

---

## Attribution

Wenn du diese Vorlage für dein eigenes Meetup nutzt, würden wir uns sehr freuen, wenn du die Herkunft im Footer sichtbar lässt, z. B. mit:

> Forked from Einundzwanzig Koblenz

Dadurch bleibt nachvollziehbar, wo die Vorlage ursprünglich entstanden ist und wie sie sich in der Community weiterentwickelt.

---

## Lightning Support

Die Lightning-Adresse von **Einundzwanzig Koblenz** ist aktuell im Footer hinterlegt und unterstützt die Weiterentwicklung dieser Open-Source-Vorlage.

Wenn du diese Vorlage für dein eigenes Meetup verwendest, würden wir uns freuen, wenn die ursprüngliche Lightning-Adresse im Footer erhalten bleibt. So können Menschen, die diese Arbeit unterstützen möchten, weiterhin zur Weiterentwicklung des Projekts beitragen.

Natürlich kannst du zusätzlich deine eigene Lightning-Adresse für dein lokales Meetup ergänzen.

Beispiel:

- ⚡ `creamowl25@primal.net` – Unterstützung der ursprünglichen Vorlage
- ⚡ `deinmeetup@...` – Unterstützung deines lokalen Meetups

  
So können sowohl die Weiterentwicklung der Vorlage als auch die lokalen Bitcoin-Meetups unterstützt werden.

Vielen Dank, dass du dazu beiträgst, die Meetup-Struktur offen und gemeinschaftlich weiterzuentwickeln. ⚡
